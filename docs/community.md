## Community Guidelines

We follow the [Contributor Covenant Code of Conduct](https://github.com/semantica-agi/semantica/blob/main/CODE_OF_CONDUCT.md). Our community values:

- **Respect** — Treat everyone with kindness and patience
- **Inclusion** — Welcome all backgrounds and experience levels
- **Collaboration** — Work together toward better solutions
- **Learning** — Share knowledge openly and help others grow

If you experience unacceptable behavior, open an issue with a `[CoC]` prefix. We investigate every report.

---

## Ways to Contribute

- **Code** — Fix bugs, add features, improve performance
- **Documentation** — Fix typos, improve guides, add examples
- **Testing** — Report issues, write tests, validate fixes
- **Community** — Answer questions, review PRs, share your work

See the [Contributing Guide](contributing) for full details.
